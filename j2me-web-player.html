📖 Инструкция по использованию

1. Использование с freej2me-web

  • Разместите freej2me-web на своём хостинге (GitHub Pages, свой сервер)
  • Подготовьте ZIP-файлы игр через "Export Data" в эмуляторе
  • Укажите URL в настройках: run.html?app=game_id
  • Сервер должен поддерживать Range header

2. Клавиши управления

  • ↑↓←→ — Навигация
  • Enter — OK / Action
  • F1 или Q — Левая софт-клавиша
  • F2 или W — Правая софт-клавиша
  • 0-9 — Цифровые клавиши
  • Esc — Настройки эмулятора

3. Интеграция в Tilda

  • Загрузите этот HTML-файл на свой хостинг
  • В Tilda добавьте блок "HTML-код" (T228)
  • Вставьте iframe: <iframe src="YOUR_URL" width="400" height="750"></iframe>

4. Решение проблем

  • CORS ошибки — разместите файлы на одном домене или настройте CORS заголовки
  • Игра не загружается — проверьте консоль браузера (F12)
  • Медленная работа — попробуйте другой размер экрана
